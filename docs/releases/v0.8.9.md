# Quillan v0.8.9

Classification: **release note**.

Quillan v0.8.9 is a pre-1.0 breaking release built for PDS Core 0.5
(`pds-core>=0.5,<0.6`; acceptance baseline 0.5.0) and CPython 3.11 through
3.14. It completes the PDS2 route-locator architecture and installed
`paper_data_suite.modules` profile discovery.

Printable generation creates immutable generation, output-artifact, issuance,
response-page, and Core route records, with exactly one route per physical page.
Continuation pages share an issuance but have distinct page and route identities.
Regeneration creates new immutable identities while old locators remain valid.

Core retains each selected scan source once and dispatches pages independently.
Quillan persists immutable page observations and routed evidence with retained
source provenance, then assembles submissions from issuance and page records—not
filenames. Teacher-controlled review includes Core routing review, distinct
Quillan post-dispatch review, page management, minimum requirements, review
units, Focus Standard observations and ratings, feedback, private notes,
assignment-local reports, dashboard schema version 2, and student review status
schema version 1. Plain-paper submissions remain intentionally free of digital
page, route, issuance, observation, and evidence identities.

All assignment data lives below
`classes/<class_id>/modules/quillan/work/<assignment_id>/`. Direct commands and
the compact, low-information-density menu expose the completed workflows.
Package installation provides `quillan` and the Quillan module entry point.
PDF scan intake requires Poppler in addition to the Python dependencies.

## Breaking changes

There is no compatibility for PDS1 QR payloads, parsing, generation, or route
planning; unqualified assignment directories; old text-oriented submission
metadata; filename-authoritative discovery; retired compatibility modules;
legacy migration; or read-only legacy access. Old development workspaces may be
discarded. Unsupported schema text is rejected without recovering or printing
private suffix identity.
